## 🎯 Overview

AirWatch Station is a team-based science project that demonstrates how technology can solve real environmental monitoring challenges. The system collects sensor data from IoT devices, stores it in the cloud, visualizes it through interactive dashboards, and provides AI-powered health insights.

### Project Vision

Create a professional, working demonstration of smart environmental monitoring that:
- **Measures** air quality, temperature, and humidity in real-time
- **Logs** data to the cloud with 7-day retention
- **Visualizes** trends through beautiful dashboards
- **Analyzes** data for anomalies and safety alerts
- **Advises** users through an AI-powered health assistant

## ✨ Features

### Core Functionality
- ✅ **Real-time Monitoring** - Live sensor data updates every 30 seconds
- ✅ **Cloud Data Storage** - 7+ days of historical data retention
- ✅ **Interactive Dashboards** - Beautiful charts and graphs
- ✅ **Health Alerts** - Automated notifications for unsafe conditions
- ✅ **AI Assistant** - Natural language queries about air quality
- ✅ **Mobile Responsive** - Works on desktop, tablet, and phone

### Technical Features
- 🔐 **Secure Communication** - HTTP/MQTT protocols
- 💾 **Persistent Storage** - Docker volumes for data persistence
- 📊 **Data Analytics** - Historical trends and patterns
- 🎨 **Modern UI** - Built with React, TypeScript, Tailwind CSS
- 🐳 **Containerized** - Easy deployment with Docker Compose
